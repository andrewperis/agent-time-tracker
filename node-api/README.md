# Node API Skeleton

This folder contains a minimal Express-based web API skeleton.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the server:

   ```bash
   npm start
   ```

3. During development, start with auto-reload:

   ```bash
   npm run dev
   ```
